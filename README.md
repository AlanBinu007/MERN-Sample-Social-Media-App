
# Sample Social Media Application using MERN Stack


## Tech We Used

- ReactJs
- Firebase Hosting
- Firebase Auth
- Firebase Storage
- React-Dom
- React Redux
- MongoDB
- ExpressJs
- NodeJs

## Features

- Login and Logout
- Like the profile
- Upload Profile photo
- Make post
- Delete post
- Like the post
- Neat and clean UI

## Steps to run in your machine

#### Run the following commands
```
cd client
npm i
npm run start

cd server
npm i
node index.js
```
